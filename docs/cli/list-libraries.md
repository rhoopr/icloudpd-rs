# `--list-libraries`

List all available iCloud Photos libraries and exit.

## Usage

```sh
icloudpd-rs -u my@email.address --list-libraries
```

## Details

- **Default**: `false`
- **Type**: Boolean flag

Lists both private and shared libraries available in your iCloud account, then exits without downloading.
